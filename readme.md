Ακολουθώντας τις βέλτιστες πρακτικές στη διαχείριση αποβλήτων, το Τμήμα Ψηφιακών Συστημάτων εφαρμόζει από τις αρχές του 2026 διαδικασίες διαλογής στην πηγή για τα απορρίμματά του. Στο πλαίσιο αυτό, το Τμήμα σε επαφή με αρμόδιους φορείς, δημιούργησε ένα κεντρικό σημείο συλλογής ανακυκλώσιμων υλικών.

Καθοριστική ήταν η συμβολή των φοιτητών, οι οποίοι σχεδίασαν και υλοποίησαν τον παρόντα δικτυακό τόπο. Μέσω της πλατφόρμας αυτής, κάθε ενδιαφερόμενος μπορεί να ενημερωθεί για τα είδη των υλικών και τις ακολουθούμενες διαδικασίες ανακύκλωσης εντός του Τμήματος.
